## âœ¨ What is this pack?

**Ramenlikkers** (Dutch for *window lickers*) is a curated Fabric modpack for our beautiful, chaotic friend group.

This pack will NOT:
- Melt your PC  
- Require 32GB RAM  
- Summon Herobrine (intentionally)  

It *will*:
- Boost your FPS  
- Add proximity voice chat  
- Make Shulker boxes actually readable  
- Give you HUD tools like MiniHUD, Peek & Status  
- Improve visuals with Raised, Shade & Saturation
